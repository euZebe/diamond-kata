# Diamond kata

## keywords
#TDD #kata #sofwareCraftsmanship

## objective
given a letter (let's say 'F'), should return the following letter diamond:

         A
        B B
       C   C
      D     D
     E       E
    F         F
     E       E
      D     D
       C   C
        B B
         A
